### SRS (間隔重複) 算法

| 動作 | 下次複習間隔 |
|---|---|
| ❌ 不會 | 1 天後 |
| ✅ 會了 (第 1 次) | 2 天後 |
| ✅ 會了 (第 2 次) | 4 天後 |
| ✅ 會了 (第 3 次) | 8 天後 |
| ✅ 會了 (第 4 次+) | 依序倍增，上限 30 天 |

**這是 Anki、Quizlet 都在用的認知科學方法**。

## 🚨 重要:為什麼你前面說看不到偷看功能?

### 根本原因:**Service Worker 快取**

PWA 的特性是把 APP 存在你手機/瀏覽器裡，下次打開**優先用本地版本**，所以 GitHub 更新了、Cloudflare 重新部署了，你看到的還是舊的。

### v0.5 的 SW 改成 Network-first 策略
以後只要網路通，一定會先抓最新版，不再卡快取。這次升級後就不會再遇到這問題。

### 但這次升級還是要強制清一次

因為你現在的 SW 是 v0.3/v0.4 舊邏輯，**要先讓它被換掉**才會啟用新策略。

## 🔧 升級步驟(這次特別講清楚怎麼強制更新)

### Step 1: push 到 GitHub

1. 下載 v5 所有檔案，覆蓋到 `D:\eng-flash\`
2. GitHub Desktop → commit message: `v0.5: 單字庫閃卡複習 + 手動新增單字`
3. `Commit to main` → `Push origin`

### Step 2: 確認 Cloudflare 部署完成

1. 到 Cloudflare Dashboard → Workers & Pages → 你的 Pages 專案
2. Deployments 頁，**等最新一筆變綠色 Success** (1-2 分鐘)

### Step 3: 強制清除舊快取 ⚠️ 最關鍵

**選擇對應你的裝置，照做**:

#### 🖥️ 電腦 Chrome / Edge
1. 打開你的網址
2. 按 **`F12`** 打開開發者工具
3. 上方 tab 切到 **Application** (應用程式)
4. 左側 **Storage** 區 → 點 **Clear site data** (清除網站資料)
5. 左側 **Service Workers** → 點 **Unregister** (取消註冊)
6. 關閉開發者工具
7. 按 **`Ctrl + Shift + R`** (Windows) 或 **`Cmd + Shift + R`** (Mac) 強制重整
8. 應該看到最新版

#### 📱 iPhone Safari (最徹底方法)
1. **長按桌面上的 Eng Flash icon** → 「移除 App」→「從主畫面移除」
2. 打開 **設定 App** → 向下捲到 **Safari**
3. 點 **進階** → **網站資料**
4. 搜尋 `pages.dev` → 向左滑 → **刪除**
5. 同樣搜尋 `workers.dev` 再刪一次
6. 回到 Safari，打開你的網址
7. 點下方分享 → **加入主畫面**
8. 用新 icon 打開，應該看到最新版

#### 📱 Android Chrome
1. 長按桌面 Eng Flash → 移除
2. Chrome → 右上三個點 → **設定** → **隱私權和安全性**
3. **清除瀏覽資料** → 勾「**Cookie 和網站資料**」+「**快取圖片和檔案**」
4. 時間範圍選「**不限時間**」→ **清除資料**
5. 重開網址 → 右上三個點 → **加到主畫面**

### Step 4: 驗證 v0.5 成功

打開 APP，檢查這三點:

| 檢查項 | 應該看到 |
|---|---|
| 首頁主題數 | **6 個**(有 🍽️ 消費生活) |
| 練習頁的中文題目卡 | 下方有 `👁️ 點我偷看答案(會加入複習)` |
| 單字庫頁頂部 | 有 **[🎯 開始複習] [➕ 新增單字]** 兩個按鈕 |
| 設定頁底部版本號 | **v0.5** |

**四個都對 = 升級完全成功** ✅

如果只看到部分對、部分錯，就是快取沒清乾淨，**重做 Step 3**。

## 🐛 如果一直卡舊版

### 終極解法:換個瀏覽器測試

1. 電腦用 **無痕視窗** (Ctrl+Shift+N) 開網址 → 應該看到最新版
2. 如果無痕也是舊的 → **Cloudflare 部署有問題** → 回 Dashboard 看 Deployments 狀態
3. 如果無痕是新的，但原本瀏覽器舊 → **你的瀏覽器快取太頑固** → 徹底清除瀏覽器資料

### 還是不行?
